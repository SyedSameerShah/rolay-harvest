## How to Download from Pexels:

1. Visit the Pexels link for each video type
2. Select a high-quality video that matches the theme
3. Download the "HD" or "Full HD" version
4. Compress if needed to keep under 2MB using tools like HandBrake or online compressors
5. Rename to match the filename above
6. Place in this `/public/videos/` directory
