# Logo Setup Instructions

## Quick Start

The logo image you provided needs to be saved as `public/logo.png` in the project.

### Steps:

1. **Save your logo image**:
   - Right-click on your logo image
   - Save it as `logo.png`
   - Place it in the `public/` folder of this project

2. **Recommended specifications**:
   - Format: PNG with transparent background
   - Size: 300x300px or similar aspect ratio
   - File name: `logo.png` (exactly)
   - Location: `public/logo.png`

### Where the logo is used:

- ✅ Header navigation (responsive sizes)
- ✅ Hero section on homepage  
- ✅ Footer

### File structure:
```
rolay-harvest/
├── public/
│   └── logo.png  ← Place your logo here!
├── src/
└── ...
```
